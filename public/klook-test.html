Klook 위젯 단독 테스트 (React/사이트 코드 없음)

이 페이지는 형이 준 Klook 코드를 "원본 그대로" 넣은 순수 HTML 입니다.

  • 여기서 배너가 뜨고 클릭이 정상 상품페이지로 가면 → Klook 코드/계정은 정상. (그렇다면 메인 사이트 통합 쪽을 더 고치면 됨)
  • 여기서도 배너가 안 뜨거나 클릭이 "Where did that page go?" → 홈으로 가면 → 이건 사이트 코드가 아니라 Klook 위젯/계정 설정 문제입니다. (Klook 제휴 대시보드에서 위젯 재발급 또는 고객지원 문의 필요)

확인 방법: 배포 후 https://(사이트주소)/klook-test.html 로 접속. 애드블록은 반드시 끄고 확인하세요.
